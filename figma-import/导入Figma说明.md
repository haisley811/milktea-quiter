# 导入到 Figma

文件：`today-no-milktea-ui.svg`

1. 打开 Figma，点击右上角的“导入”。
2. 选择本文件夹中的 `today-no-milktea-ui.svg`。
3. 导入完成后，画布上会出现三个手机界面：`Screen-Home`、`Screen-Order`、`Screen-Success`。
4. 双击任意手机界面，或在左侧图层面板展开对应分组，即可选择并调整文字、按钮、卡片、颜色和圆角。
5. 若一次只能选中整张 SVG，右键点击它，选择“取消编组”；重复此操作直到可以选中具体元素。

说明：SVG 中的主要元素使用了分组、矩形、圆形和文本。Figma 会将图形作为可编辑矢量图层导入；如果个别文字导入后显示为矢量轮廓，可直接删掉该文字并使用 Figma 的文本工具重新输入。
